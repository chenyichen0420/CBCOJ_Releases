## Basic Information

It's the newest version of CBCOJ before 2026.01.24 22:00 [UTC+8].

However, there won't be an official release of this version.

## What's wrong?

According to the [Features.md](https://github.com/chenyichen0420/CBCOJ_Releases/blob/main/Features.md), There has long been an insecure network connection between the front-end and back-end.

We are trying to fix this.

In this version, the backend were fixed with some other features implemented.

By contrast, the frontend won't be ready for a few days.

However, it's still a significant milestone for CBCOJ. Therefore, it has been assigned a major version number.
